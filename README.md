### 🧠 AI Medical Report Analyser — RAG-Powered Clinical Assistant

> **The constraint:** in healthcare, a hallucinated answer isn't just wrong — it's dangerous.

**Built around accuracy, grounding, and traceability**

- 🗄️ **Dual Pinecone index** (GENERAL_INDEX + PATIENT_INDEX) — strict session isolation, answers never cross between patients
- 🔍 **MMR retrieval** (k=10, λ=0.35) — minimises redundant chunks, maximises answer diversity
- 🌡️ **Gemini 1.5 Flash at temperature=0.2** — grounding prompts reduce hallucination in medical context
- 📏 **Evaluation framework** — faithfulness score, hallucination rate, grounding rate, retrieval latency, LLM latency per session
- 🏷️ **Full traceability** — every answer tagged `[patient]`, `[helpbook]`, or `[web]`
