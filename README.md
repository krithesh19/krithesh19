### SaaS Revenue Intelligence Platform

> Most data pipelines stop at a dashboard. This one doesn't.

```
Azure Blob Storage
      ↓
  Snowflake  (RAW → STAGING → GOLD)
      ↓
  dbt Core   (5 GOLD models · 14 tests passing)
      ↓
  Apache Airflow  (nightly 02:00 UTC · idempotent · Slack alerts)
      ↓
  Databricks PySpark  (cohort retention · churn · LTV · MRR trend)
      ↓
  Streamlit (11 pages) + LangChain AI SQL Agent (Groq LLaMA 3.3 70B)
```

- 🤖 **AI SQL agent** — business users query Snowflake in plain English, zero SQL required
- ⏰ **Nightly Airflow DAG** — idempotent loads, exponential backoff, Slack success/failure alerts
- 📊 **Metrics tracked** — MRR, ARR, Churn, NRR, LTV, CAC, Quick Ratio, Rule of 40
- 🔬 **Databricks PySpark** — cohort retention heatmap, churn rate, LTV estimation, MRR trend

### AI Medical Report Analyser — RAG-Powered Clinical Assistant

> In healthcare, a hallucinated answer isn't just wrong — it's dangerous. Built around that constraint.

- 🗄️ **Dual Pinecone index** (GENERAL_INDEX + PATIENT_INDEX) — strict session isolation, no answer ever crosses between patients
- 🔍 **MMR retrieval** (k=10, λ=0.35) — minimises redundant chunks, maximises context diversity
- 🌡️ **Gemini 1.5 Flash at temperature=0.2** — grounding prompts engineered to prevent hallucination
- 📏 **Evaluation framework** — faithfulness, hallucination rate, grounding rate, retrieval latency, LLM latency per session
- 🏷️ **Full traceability** — every answer tagged `[patient]`, `[helpbook]`, or `[web]`
